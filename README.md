# 📂 File Organizer (Python)

A simple Python script that automatically organizes files in a folder into different categories based on their file type.

---

## 🚀 Features

- 📁 Organizes files into folders (Images, Documents, PDFs, Others)
- 🔍 Detects file types automatically
- 🧹 Cleans messy folders
- ⚡ Works for any folder (user input)
- 🛠️ Creates folders if they don’t exist

---

## 🛠️ Technologies Used

- Python
- os module
- shutil module

---

## ▶️ How to Run

1. Make sure Python is installed  
2. Download or clone this repository  
3. Run the script:

```
python file_organizer.py
```

4. Enter the folder path you want to organize  

---

## 📂 Example

Before:
```
Downloads/
   photo.jpg
   notes.txt
   file.pdf
   song.mp3
```

After:
```
Downloads/
   Images/
   Documents/
   PDFs/
   Others/
```
